# SimplePDFMerger

**SimplePDFMerger** は、Python 3.11 で動作するシンプルかつ直感的なデスクトップ向けPDF結合アプリケーションです。
ドラッグ＆ドロップによる簡単な操作で、複数のPDFファイルを一つのファイルにまとめることができます。

## 概要 (Description)

このツールは、GUI（グラフィカル・ユーザー・インターフェース）を使用して、複雑なコマンド操作なしにPDFの結合を行うために開発されました。
ファイルリストへの追加、並び替え、削除を視覚的に行い、ボタン一つで結合処理を実行します。

主な解決課題：

  * コマンドラインを使わずにPDFを結合したい。
  * 結合する前にページの順番を自由に入れ替えたい。
  * 直感的なドラッグ＆ドロップ操作でファイルを管理したい。

## 機能 (Features)

ソースコードに基づく主な機能は以下の通りです。

  * **ドラッグ＆ドロップ対応:** アプリケーションウィンドウにPDFファイルを直接ドロップして追加可能。
  * **リスト管理:**
      * **上へ / 下へ:** 選択したファイルの結合順序を変更。
      * **削除:** 不要なファイルをリストから除外。
      * **ファイル参照:** ダイアログボックスからのファイル選択もサポート。
  * **結合実行:** `PyPDF2` を使用した高速なPDF結合処理。
  * **バックグラウンド処理:** 結合処理は別スレッドで行われるため、処理中にアプリがフリーズしません。
  * **安全設計:** 最大15ファイルまでの制限機能付き（過剰な負荷を防止）。

## 前提条件 (Requirements)

  * **Python 3.11**
  * OS: Windows / macOS / Linux (Tkinterが動作する環境)

## インストール (Installation)

このプロジェクトをローカル環境で実行するには、以下の手順に従ってください。

1.  **リポジトリのクローン（またはコードの保存）**
    ソースコードを任意のディレクトリに保存します（例: `main.py`）。

2.  **依存ライブラリのインストール**
    `pip` コマンドを使用して、必要なライブラリをインストールします。

    ```bash
    pip install tkinterdnd2 PyPDF2
    ```

    > **Note:** 標準ライブラリの `tkinter`, `os`, `threading` はインストールの必要はありません。

## 使い方 (Usage)

1.  **アプリケーションの起動**
    ターミナルで保存したPythonファイルを実行します。

    ```bash
    python main.py
    ```

2.  **ファイルの追加**

      * 「ここにPDFファイルをドラッグ＆ドロップ」と書かれたエリアにファイルをドロップするか、「ファイルを参照して追加」ボタンをクリックしてPDFを選択します。
      * **制限:** 一度に追加できるファイル数は最大15ファイルです。

3.  **順序の調整**
    リスト内のファイルを選択し、「↑ 上へ」または「↓ 下へ」ボタンを押して結合順序を調整します。

4.  **結合の実行**
    「PDFを結合」ボタン（水色のボタン）をクリックします。保存先ダイアログが表示されるので、ファイル名を指定して保存してください。
